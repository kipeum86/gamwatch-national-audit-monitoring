영상 분석 요청

        
      

      

        
유튜브 영상의 자막을 복사하여 붙여넣으면, AI가 게임 산업 관련 발언을 분석합니다.

        

          
          

            
            
          

        

        

          

            
            
          

          

            
            
          

        


        
        

          
          

          
          

        


        
        

          YouTube에서 자막 복사하는 방법
          

            
    
              
  1. 
                YouTube 영상 페이지 열기
                
    분석할 영상의 YouTube 페이지로 이동합니다.
    
              
    2. 
              
  2. 
                영상 아래 "...더보기" 클릭
                
    영상 제목/설명 아래의 ...더보기 버튼을 클릭합니다.
    
              
    3. 
              
  3. 
                "스크립트 보기" 클릭
                
    펼쳐진 영역 하단에 스크립트 보기 버튼이 있습니다.
    
              
    4. 
              
  4. 
                전체 선택 후 복사
                
    스크립트 패널에서 Ctrl+A (Mac: Cmd+A)로 전체 선택 후 Ctrl+C로 복사합니다.
    
              
    5. 
              
  5. 
                위 텍스트 영역에 붙여넣기
                
    Ctrl+V로 붙여넣으면 자동으로 글자 수가 표시됩니다.
